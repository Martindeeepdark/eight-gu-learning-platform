# SDD 开发文档

本目录包含按照 Spec-Driven Development (SDD) 方法论编写的项目规范文档。

## 文档说明

### 阶段文档

1. **01-constitution.md** - 项目立宪
   - 核心使命和价值观
   - 成功指标和约束条件
   - 质量标准和可持续发展承诺

2. **02-specify.md** - 规格定义
   - 功能规格（用户管理、知识库、学习路径等）
   - 技术规格（后端、前端、数据库）
   - 接口规格和验收标准

3. **03-clarify.md** - 需求澄清
   - 需求优先级分析
   - 技术决策说明
   - 风险分析和应对措施

4. **04-plan.md** - 技术规划（待生成）
   - 架构设计
   - 数据模型
   - API 契约
   - 实施计划

5. **05-tasks.md** - 任务分解（待生成）
   - 可执行任务清单
   - 依赖关系和并行任务
   - 验证标准

## 文档更新

所有文档在开发过程中会持续更新，确保与实际实现保持一致。
